📊 Customer Data Analysis Project📝 Project OverviewThis project focuses on cleaning and analyzing retail customer data to uncover actionable business insights. It involves processing transaction history and customer demographics to understand purchasing patterns.  🛠️ Tech StackLanguage: Python  Libraries: Pandas, NumPy, Openpyxl  Environment: Jupyter Notebook  🔑 Key Tasks PerformedData Wrangling: Cleaned and merged Customer_Master_Data and Customer_Transactions datasets.  Feature Engineering: Converted date formats and handled missing values for accurate analysis.  Segmentation: Grouped data by city and gender to identify high-value customer segments.  .
